# Task Blog Frontend

## App URL

https://task-blog-greencast.web.app/ (hosted on Firebase Hosting)

## Run locally

In the project directory, run:

- Install dependencies

  ```
  npm install
  ```

- Start app

  ```
  npm run start
  ```

- Build the app
  ```
  npm run build
  ```
